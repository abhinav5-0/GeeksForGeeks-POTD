- **Languages Used:** C++/Python.

- # Problem: Example Problem

## Problem Statement:
Given two integers `a` and `b`, return their sum.

---

## Approach:
1. Take input for `a` and `b`.
2. Add the two numbers.
3. Return the result.

---

## Code:
```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;
    cout << a + b << endl;
    return 0;
}
